
--MaxOne Problem for Eiben & Smith Textbook 3-7--

Use terminal to compile this python file, type "python MaxOneProblem.py"
It will generate two kinds of output file based on your run_time value which should be manually changed
The value of run_time = 1 is for first question.
The value of run_time = 10 is for second question.
Based on the output file, we can use excle to plot diagrams required by original problem.
